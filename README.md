To run the project:
1. ``` git clone https://github.com/danaetroupe/tetris-2.0```
2. ``` cd tetris-2.0 ```
3. ``` npm i live-server```
4. ``` live-server```
5. Play!

**Controls:**
Left/Right Arrows -> Move Left/Right
Up Arrow          -> Rotate
Down Arrow        -> Soft Drop 
Shift             -> Hard Drop
'C'               -> Store/Swap Piece

When complete, enter your initals to save your score.
